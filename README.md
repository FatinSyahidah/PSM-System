## About This Project

This repository is a fork of a group project developed for the Software Engineering Practice course.

The system is a web-based **PSM Management System** designed to manage final year project processes including meetings, supervisor hunting, proposal submission, logbook updates, title management, and inventory usage.

## My Contribution

My responsibility in this project focuses on the **Inventory Usage Module**, which includes:

- Requesting inventory items
- Viewing request status
- Lecturer approval or rejection of inventory requests
- Recording pick-up details
- Managing inventory usage records

The module was implemented following the MVC architecture using Laravel framework.
